Разработка сайта фабрики печенья «AMI»
Курсовой проект по дисциплине «Основы web-программирования».

🎨 Проектирование и дизайн
На данном этапе ведется разработка визуальной концепции и структуры интерфейса.

Ссылка на проект в Figma:(https://www.figma.com/design/uQfFuLPtW0W8w6f5o4m5j5/%D0%9A%D1%83%D1%80%D1%81%D0%B0%D1%87-%D0%BC%D0%B0%D0%BA%D0%B5%D1%82-%D0%B8%D0%B7%D0%BC%D0%B5%D0%BD%D0%B5%D0%BD%D0%BD%D1%8B%D0%B9?node-id=2837-2756&t=sf5rHCLFl60JDvFR-0)

🛠 Технологический стек
Frontend: HTML5, CSS3, JavaScript (Native)
Инструменты: Figma, GitHub, VS Code
Данные: JSON-server / LocalStorage
📝 Основные функции
Адаптивный каталог продукции.
Система фильтрации и поиска по видам печенья.
Личный кабинет и корзина покупателя.
Поддержка двух языков (RU/EN).
Тёмная/светлая темы и версия для слабовидящих.
